To bypass Apple Codesigning, run:

xattr -d com.apple.quarantine [YOUR PATH TO worm-world.app]


The app doesn't come pre-populated with C. Elegans genetic data, so to get started, you'll likely want to download some test data.

Starting Dataset:

The unzipped CSV/TSV files can be uploaded to the app's database using the "Import" button on the corresponding Data Manager page.
